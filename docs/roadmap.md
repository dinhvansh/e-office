# Development Roadmap

## Phase 1 – Core Platform
- Authentication
- Tenant management
- Document upload
- PDF viewer integration
- Signing fields UI
- Basic signing workflow

## Phase 2 – Advanced Features
- OTP signing
- Audit trail
- Template system
- QR verification
- Webhooks
- API integrations

## Phase 3 – SaaS Layer
- Billing integration (Stripe/Momo)
- Usage limits
- Subscription management
- Admin panel for SaaS Owner

## Phase 4 – On-Premise Edition
- License activation server
- Offline activation file
- Docker installer
- Self-host admin console

## Phase 5 – Enterprise Features
- PKI digital signing
- HSM integration
- LDAP/SSO (Azure AD, Google Workspace)
- Enterprise audit compliance
