# Historical archive

Everything below this directory is retained only as a historical engineering
record. It is not deployment guidance, supported API documentation, or a
source of production credentials. Names, `acme.local` examples, test URLs, and
commands may be stale or intentionally fictitious.

Use the root README, `.env*.example` files, and the current deployment guides
for a new installation. Do not copy credentials, tokens, hostnames, or paths
from archived material into any environment.
