## 📋 Cần Cài Trước

### 1. Node.js (5 phút)
- Tải: https://nodejs.org/
- Chọn phiên bản **LTS** (v20.x)
- Cài đặt → Next → Next → Finish

### 2. Docker Desktop (10 phút)
- Tải: https://www.docker.com/products/docker-desktop/
- Cài đặt → Khởi động Docker Desktop
- Đợi icon màu xanh ở góc dưới phải màn hình

## 🆘 Gặp Lỗi?

### "Docker is not running"
→ Mở Docker Desktop và đợi nó sẵn sàng

### "Port already in use"
→ Chạy: `.\stop-all.ps1` rồi chạy lại

### "Cannot find module"
→ Xóa thư mục `node_modules` và chạy lại script
